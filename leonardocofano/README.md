# Leonardo Cofano — Personal Website

Sito personale realizzato come progetto finale del Master in UX/UI Design & AI presso Start2Impact.

## Tecnologie utilizzate

- HTML5 semantico
- CSS / **SCSS** (Sass) — compilato in `assets/css/style.css`
- Bootstrap 5 (grid system)
- JavaScript vanilla
- Google Fonts (Playfair Display, DM Sans, DM Mono)

## Struttura

```
leonardocofano/
├── index.html          # Homepage
├── cv.html             # Curriculum Vitae (HTML)
├── contatti.html       # Pagina contatti con form
├── assets/
│   ├── scss/
│   │   └── style.scss  # Sorgente SCSS
│   ├── css/
│   │   └── style.css   # CSS compilato
│   ├── js/
│   │   └── main.js     # Script JS
│   └── img/
│       ├── foto.jpg    # Foto personale
│       └── favicon.svg # Favicon
└── README.md
```

## Funzionalità

- ✅ Navbar sticky e responsive con hamburger menu (mobile)
- ✅ Layout con Flexbox e CSS Grid
- ✅ Animazioni scroll (Intersection Observer)
- ✅ Form di contatto con validazione e feedback visivo
- ✅ Open Graph tags per social sharing
- ✅ Favicon personalizzata
- ✅ Sito 100% responsive
- ✅ CSS scritto in SCSS/Sass
- ✅ Bootstrap 5 (grid)

## Come compilare il SCSS

```bash
sass assets/scss/style.scss assets/css/style.css --watch
```
